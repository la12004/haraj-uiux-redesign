# Haraj App UI/UX Redesign

## About the Project

This project focuses on redesigning the Haraj mobile application to create a more intuitive, user-friendly, and visually appealing experience.

Through user research and usability analysis, several pain points were identified, including navigation difficulties, interface complexity, and inefficient user flows. The redesign aims to simplify interactions, improve accessibility, and enhance the overall user experience.

## Project Highlights

✨ Modern and clean interface

🧭 Improved navigation experience

🚀 Enhanced usability and accessibility

🎨 Consistent visual design system

📱 High-fidelity mobile prototype

## Redesigned Screens

* Login Screen
* Home Screen
* Ad Creation Screen
* Product Details Screen
